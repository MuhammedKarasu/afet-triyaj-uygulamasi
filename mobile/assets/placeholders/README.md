# AfetSaha mobil görsel yerleşimleri

Bu klasör daha sonra eklenecek nihai marka dosyaları için ayrılmıştır.

| Kullanım | Önerilen kaynak | Yerleşim |
|---|---:|---|
| Uygulama logosu | 1024 × 1024 PNG | `../images/icon.png` |
| Splash ikonu | kare ve transparan PNG | `../images/splash-icon.png` |
| Splash kompozisyonu | 1290 × 2796 | `app/index.tsx` içindeki splash yerleşimi |
| Giriş illüstrasyonu | 1600 × 2000 | `app/login.tsx` içindeki illüstrasyon alanı |

Expo ikonu veya splash dosyası değiştirildikten sonra Metro sunucusunu `npx expo start --clear` ile yeniden başlatın.
